#  Generador de Texto con IA

##  Descripción del proyecto

Este proyecto es una herramienta basada en Inteligencia Artificial que permite generar diferentes tipos de contenido automáticamente utilizando modelos de lenguaje (GPT-2 a través de la librería Transformers).

La aplicación cuenta con una interfaz web sencilla creada con Gradio y permite al usuario interactuar con la IA sin necesidad de conocimientos técnicos.

---

##  Funcionalidades

La herramienta permite:

-  Generar textos cortos a partir de un prompt.
-  Crear descripciones de productos.
-  Generar ideas para proyectos.
-  Crear resúmenes automáticos de textos.
-  Exportar resultados generados.

---

## 🛠️ Tecnologías utilizadas

- Python 🐍
- Transformers (Hugging Face)
- GPT-2 (modelo de lenguaje)
- Gradio (interfaz web)
- PyTorch
